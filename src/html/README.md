# HTML 測試頁面說明

本目錄包含3D Bin Packing系統的測試頁面。

## 文件說明

### `test-pages.html`
測試頁面的入口頁面，提供所有測試功能的導航。

**訪問方式：**
- 從主頁點擊 "Test Pages" 按鈕
- 直接訪問 `src/html/test-pages.html`

### `simple-test.html`
基本的3D場景測試頁面，用於驗證Three.js和物理引擎的基本功能。

**功能：**
- 基本的3D場景渲染
- 物理引擎測試
- 物件創建測試

### `test-scene.html`
3D場景測試頁面，測試場景的渲染和物件管理功能。

**功能：**
- 場景管理測試
- 物件互動測試
- 渲染性能測試

## 使用方法

1. 啟動主應用程式
2. 點擊左側工具列的 "Test Pages" 按鈕
3. 選擇要測試的功能頁面
4. 進行相應的測試

## 注意事項

- 測試頁面需要主應用程式的JavaScript模組
- 確保所有依賴文件路徑正確
- 測試完成後可以返回主頁面繼續使用
